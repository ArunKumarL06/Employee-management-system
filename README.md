# Employee-management-system
Full Stack Employee Management System built with React, Vite, Spring Boot 3, Java 21, JWT Authentication, MySQL, and Swagger API Documentation.
# Enterprise Employee Management System

A modern full-stack Employee Management System designed for enterprise environments. The application provides secure employee administration, department management, attendance tracking, leave management, payroll processing, and role-based access control.

## Features

### Authentication & Security

* JWT Authentication
* User Registration & Login
* Role-Based Access Control (Admin / Employee)
* Spring Security Integration
* Password Encryption using BCrypt

### Employee Management

* Create Employee Records
* Update Employee Information
* Delete Employees
* Search & Filter Employees
* Employee Profile Management

### Department Management

* Create Departments
* Update Departments
* Delete Departments
* Department Assignment

### Attendance Management

* Employee Check-In
* Employee Check-Out
* Attendance Tracking
* Attendance Reports

### Leave Management

* Apply Leave Requests
* Approve/Reject Leave Requests
* Leave Balance Tracking

### Payroll Management

* Payroll Generation
* Salary Distribution
* Payroll History

### API Documentation

* Swagger/OpenAPI Integration
* Interactive API Testing

## Technology Stack

## Getting Started

### Backend

mvn spring-boot:run

### Frontend

npm install
npm run dev
